My Work

GaliGo

GaliGo is a company website that I developed from scratch for 3D Aerospace, a young tech startup that I interned with for 6 months. The website serves as a showroom for one of the company's brands that are set to launch in the near future.

My role in this project extended well beyond leading all web development activities, as I was very involved in the commercial development and design of the brand itself too.

GaliGo was built using pure HTML and JavaScript paired with SCSS as a CSS preprocessor, with the backend being developed in PHP.

IP Address Tracker

This API-based tool allows user to locate any IP address or domain, including their own, on an interactive map. Users can search for any IP address and provided it is valid, will be able to view more detailed information about that address and its geographical location.

The application is developed in HTML, CSS, and JavaScript, paired with the Ipify and Mapbox APIs.

Shortly

Shortly is another API-based project with the purpose of helping users shorten annoyingly long links and save the shortened links for the future.

The project uses HTML, CSS, and JavaScript in their purest form and data storage occurs with the help of localStorage.
